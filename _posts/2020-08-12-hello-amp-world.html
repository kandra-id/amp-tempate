Accelerated Mobile Page atau AMP, merupakan sebuah gagasan dari raksasa mesin pencari Google agar sebuah halaman web dapat diunduh dengan cepat melalui perangkat mobile. Bagi digital marketer yang mencari pelanggan baru, membuat sebuah web valid AMP akan sangat bermanfaat.

Ini langkah-langkah mengkonversi sebuah web template AMP yang didapat di laman amp.dev agar bisa disajikan dalam Github Page atau menggunakan Platform CMS Jekyll.

  1. Unduh template yang diinginkan dari laman amp.dev
  2. Install Chrome Web Server
